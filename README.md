# WeightedAverages

## Description (EN) ##
Personnal exercice to practice :
- Responsive Design
- React Library
- Formatting the display of a number

=> The command
`+parseFloat(resultW).toFixed(6)`
allows 6 digits maximum after the decimal, but only if necessary.

=> Examples :
| number | display |
| :- | :- |
| 1 | 1 |
| 1.2 | 1.2 |
| 1.234567 | 1.234567 |
| 1.23456789 | 1.234568 |
